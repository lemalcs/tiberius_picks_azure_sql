# Tiberius picks Azure SQL

This is a library Rust application to connect to an Azure SQL database using this authentication methods:
- SQL Server authentication
- Microsoft Entra authentication with a service principal (application tokens).

You can also read the [complete journey to create this Rust application](https://medium.com/@lemalcs/connecting-rust-to-azure-sql-without-losing-your-mind-33ea2863bc86).
